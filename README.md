# Telecom Customer Churn Prediction

## Overview

This project focuses on predicting customer churn in the telecom industry using machine learning. It also includes a Power BI dashboard for visualizing customer behavior and churn patterns.

## Files

- `telecom_churn_prediction.ipynb` – Jupyter Notebook containing data preprocessing, exploratory data analysis, feature engineering, model training, and evaluation.
- `Telco.csv` – Dataset used for model development.
- `telecom_churn_analysis.pbix` – Interactive Power BI dashboard.
- `BI_report.pdf` – Dashboard report and project summary.
- `requirements.txt` – Python dependencies.
- `LICENSE` – MIT License.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Power BI
- Jupyter Notebook

## Project Workflow

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Model Training
- Model Evaluation
- Dashboard Development

## Dataset

The dataset contains customer demographic information, service details, billing information, contract type, internet service, payment method, tenure, monthly charges, total charges, and churn status.

## Installation

Clone the repository:

```bash
git clone https://github.com/punitd-source/Telecom-Customer-Churn-Prediction.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Open the notebook:

```bash
jupyter notebook telecom_churn_prediction.ipynb
```
